PROJET BLACKJACK

Jeu Web en ReactJS

Développement du projet

Le projet Blackjack a été développé dans le cadre d’une initiation à ReactJS. Il s’agit d’un jeu de cartes en ligne simulant une partie de blackjack jouable côté client. Ce projet a été enrichi au-delà de la version pédagogique initiale pour intégrer des fonctionnalités supplémentaires.

Le code front-end a été réalisé avec ReactJS, tandis que le back-end repose sur Node.js et une base de données MongoDB.

Il est hébergé entièrement sur Google Cloud, avec le front et le back déployés via les services Cloud Run et Cloud Storage.

Voici le lien vers le projet :
https://storage.googleapis.com/black-jack/index.html#/

Fonctionnalités principales

  • Possibilité de jouer sans être connecté
  • Inscription et connexion utilisateur
  • Historique des parties jouées
  • Système de mise et score
  • Administrateur : Suppressions des comptes utilisateurs
